# dida-2-notion

## 简介

将滴答清单（dida365.com）的待办事项同步到Notion（notion.so）。

## 功能

- 同步滴答清单的待办事项到Notion
- 支持自定义Notion数据库的属性
- 支持自定义Notion数据库的视图
- 支持自定义Notion数据库的排序
